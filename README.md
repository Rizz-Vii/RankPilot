## Quickstart

Requirements: Node.js 20+, npm

1. Install dependencies

 ```bash
 npm install
 ```

2. Start the web app (stable dev server)

 ```bash
 npm run dev-no-turbopack
 ```

Optional:

- Lint/format: `npm run lint`
- Run tests (see docs for variants): `npm test`
